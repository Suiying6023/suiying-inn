本仓库包含 Java 课程设计大作业“岁影客栈管理系统”以及教材同步练习代码。

## 岁影客栈管理系统 (suiying-inn)

`suiying-inn` 是一个以客房业务管理为背景的控制台应用程序，支持用户与管理员双角色登录、客房信息维护、客房预订流程以及基于本地文本文件的数据持久化。

### 核心功能

* **用户认证**
  * 支持管理员（Admin）与普通用户（Guest）注册及登录。
  * 用户账户信息保存在 `users.txt` 中，支持运行时密码修改。
  * 默认内置账户：
    * 管理员：`admin` / `admin`
    * 普通用户：`guest` / `guest`
* **客房预订与订单管理 (普通用户)**
  * **预订房间**：支持预约未来 7 天内的房间。系统根据输入的入住日期、退房天数自动筛选有空余的房型（大床房、双床房、套房），并计算折后总价。
  * **查看与取消订单**：支持查看预订历史，并支持未入住订单的退订。
* **客房管理 (管理员)**
  * **客房运维**：支持对客房信息进行增、删、改、查操作。
  * **订单监控**：查看所有用户的预订订单和消费金额。

### 技术栈

* **开发语言**：Java 17 (兼容 Java 8+)
* **构建工具**：Maven 3.x
* **数据存储**：本地文本文件读写 (`users.txt`)

### 快速开始

#### 环境要求
* JDK 17 或更高版本
* Maven 3.6 或更高版本

#### 编译与运行

1. 进入项目目录：
   ```bash
   cd suiying-inn
   ```
2. 编译并运行：
   ```bash
   mvn clean compile exec:java
   ```
3. 打包并运行 JAR：
   ```bash
   mvn clean package
   java -jar target/suiying-inn-1.0.0.jar
   ```
