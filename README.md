# 岁影客栈 Java 课程设计与教科书练习项目

本仓库是一个经过规范整理的 Java 课程作业与实践成果管理仓库。仓库主要包含两个部分：
1. **岁影客栈管理系统 (`suiying-inn`)**：课程设计的核心 Java CLI 应用项目，采用 Maven 构建。
2. **教材同步练习 (`textbook-exercises`)**：学习过程中的章节代码与练习题存档。

---

## 🌅 岁影客栈管理系统 (`suiying-inn`)

`suiying-inn` 是以客栈/酒店业务管理场景为背景设计的控制台应用程序。项目实现了用户与管理员双角色登录，提供了基础信息维护、客房预订流程以及简易的数据持久化。

### 🌟 核心功能

* **用户认证系统**
  * 支持管理员（Admin）与普通用户（Guest）注册及登录。
  * 用户账户信息保存在 `users.txt` 中，支持运行时密码修改。
  * 默认内置账户：
    * **管理员**：用户名 `admin` / 密码 `admin`
    * **普通用户**：用户名 `guest` / 密码 `guest`
* **客房预订与订单管理 (普通用户)**
  * **预订房间**：支持预约未来 7 天内的房间。用户可输入入住日期、退房天数，系统会自动筛选此时间段内仍有空余的房型（大床房、双床房、套房等），并计算是否需要早餐和折后总价。
  * **查看/取消订单**：查看个人预订历史，并支持未入住订单的退订（释放房间预约额度）。
* **客房管理与数据维护 (管理员)**
  * **客房运维**：支持对客房信息进行增、删、改、查（修改价格、库存等）。
  * **订单监控**：全局查看所有用户的预订订单和消费金额。

### 🛠️ 技术栈

* **核心语言**：Java 17 (兼容 Java 8+)
* **构建工具**：Maven 3.x
* **数据持久化**：本地文本文件数据流读写 (`users.txt`)
* **设计模式与逻辑**：面向对象程序设计，基于内存的对象映射（HashMap / LinkedHashMap）存储运行时订单和房态。

### 🚀 快速开始

#### 环境要求
* JDK 17 或更高版本
* Maven 3.6 或更高版本

#### 编译与运行

1. 进入项目目录：
   ```bash
   cd suiying-inn
   ```
2. 使用 Maven 直接编译并运行控制台程序：
   ```bash
   mvn clean compile exec:java
   ```
3. （可选）打包成可执行 JAR 包：
   ```bash
   mvn clean package
   java -jar target/suiying-inn-1.0.0.jar
   ```
